# Мультикалькулятор объектов

Веб-инструмент для предварительной оценки стоимости клининговых работ — после пожара, очистки металла и обработки кирпичного фасада. Предназначен для менеджеров компаний Global Cleaning и 112 Cleaning: помогает быстро получить коммерческое предложение прямо на объекте.

## Калькуляторы

| Вкладка | Что считает |
|---|---|
| Пожар | Очистка помещений после пожара: базовая ставка + надбавки за мебель, окна, мягкую мебель, технику |
| Металл | Абразивоструйная очистка металлоконструкций под покраску |
| АСТ кирпич_фасад | Очистка кирпичного фасада + опционально укрепляющий состав, гидрофобизация, расшивка швов |

## Как запустить

Проект — статический HTML без зависимостей.

**Вариант 1 — открыть напрямую:**
```
index.html → двойной клик → открывается в браузере
```

**Вариант 2 — через локальный сервер** (нужен для корректной работы `clipboard API`):
```bash
python -m http.server 8000
# затем открыть http://localhost:8000
```

**Вариант 3 — по ссылке:** инструмент задеплоен на хостинг, ссылку уточните у команды.

## Для кого

Менеджеры на выезде: вводят параметры объекта → получают ориентировочную стоимость и готовый текст для отправки клиенту.
